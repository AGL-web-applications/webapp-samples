#$.ui.unblockUI()

```

This will remove the UI mask
         
```

##Example

```
         $.ui.unblockUI()
         
```
`

##Parameters

```

```

##Returns


##Detail

$.ui.unblockUI is a shortcut to $.unblockUI from the popup plugin.

This function clears the mask that was created by $.blockUI
