#$.ui.showMask(text);

```

Show the loading mask
 
```

##Example

```
 $.ui.showMask()
 $.ui.showMask('Doing work')
 
```


##Parameters

```
[text]                        String

```

##Returns

```
undefined
```

##Detail

$.ui.showMask(text) >This will show the loading mask. You can trigger this manually for long operations.

```
$.ui.showMask("Loading...");
```
