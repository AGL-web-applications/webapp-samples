#$.ui.disableSideMenu();

```

Disables the side menu
 
```

##Example

```
 $.ui.disableSideMenu();
 
```


##Parameters

```

```

##Returns

```
undefined
```

##Detail

$.ui.disableSideMenu() will disable/remove the side menu for your app.

This function removes the special class ".hasMenu" and hides the side menu.
