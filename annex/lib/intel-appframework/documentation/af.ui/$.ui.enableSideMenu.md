#$.ui.enableSideMenu();

```

Enables the side menu if it has been disabled
 
```

##Example

```
 $.ui.enableSideMenu();
 
```


##Parameters

```

```

##Returns

```
undefined
```

##Detail

$.ui.enableSideMenu() will enable/add the side menu for your app.

This function adds the special class ".hasMenu" and enables the side menu.
