#$.ui.goBack()

```

Initiate a back transition
 
```

##Example

```
 $.ui.goBack()
 
```


##Parameters

```
[delta=1]                     Number

```

##Returns

```
undefined
```

##Detail

$.ui.goBack() sends the user back one page in the history stack.

This is the same as clicking the back button.

```
$.ui.goBack();
```
