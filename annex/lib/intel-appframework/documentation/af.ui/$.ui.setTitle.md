#$.ui.setTitle(value)

```

Set the title of the current panel
 
```

##Example

```
 $.ui.setTitle("new title");
 
```


##Parameters

```
value                         String

```

##Returns

```
undefined
```

##Detail

$.ui.setTitle(title) will change the title for the current panel.

This expects a string and will udpate the &lt;h1> tag in the header with the value.
