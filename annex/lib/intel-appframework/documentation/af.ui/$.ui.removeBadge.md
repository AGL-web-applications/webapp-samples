#$.ui.removeBadge(target)

```

Removes a badge from the selected target.
 
```

##Example

```
 $.ui.removeBadge("#mydiv");
 
```


##Parameters

```
target                        String

```

##Returns

```
undefined
```

##Detail

$.ui.removeBadge removes a badge that was created with $.ui.updateBadge

```
$.ui.removeBadge("#myListTest li");
```
